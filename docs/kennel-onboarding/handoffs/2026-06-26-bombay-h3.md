Parsing plan:
1. `fetchHTMLPage("https://bombayhash.org/")` (static Cheerio — SSR confirmed; **not** `browserRender`).
   Use `safeFetch` under the hood (the shared helper), **not** raw `fetch`.
2. Find run blocks by **heading text**, not tag: iterate headings (`h2, h3`) whose text matches
   `/\bRUN\s*#?\s*(\d{3,4})\b/i` (handles `RUN #631`, `BH3 RUN #629`, `BOMBAY HASH RUN #628`,
   `Run 627 …`). For each, capture the run number, then walk `.nextUntil(<next run heading>)` to collect
   that block's `<p>` text. (Climb to the block wrapper if headings/paragraphs aren't siblings — Astra
   wraps blocks; key on text, classes rotate.)
3. Per block, `stripHtmlTags(html, "\n")` → lines, then:
   - **Date:** the `📅`/`Date:`/`Assembly:` line → value like `Sunday, 28th June 2026` / `Sunday, 29 March 2026` /
     `Sunday, 22nd Feb 2026`. **Normalize** (strip ordinal suffix `/(\d)(?:st|nd|rd|th)\b/→$1`, drop leading
     weekday) then `chronoParseDate` (from `@/adapters/utils` — has the `D MMM YY` fast-path; it ignores the
     weekday prefix). **Year is present → NO inference.** Store **UTC noon**.
   - **Time:** `🕘`/`Time:`/`⏰`/`Assembly:` line → `9:30 AM` → `"09:30"` (`HH:MM` 24-hr string).
   - **Venue:** `📍`/`Venue:` line (or the first non-emoji line after the date) → venue name. Keep the
     bare venue; fold any address/directions lines into it or drop them. **Strip PII** (phone/payee).
   - **hares:** omit unless a real name is present; **never store `"???"`**.
   - **cost:** kennel default already on `hashCash`; emit `Event.cost` only when a run's rego differs
     (e.g. #630 `₹2100`). Optional.
   - **title:** leave **undefined** → merge synthesizes `"Bombay H3 Trail #N"`.
4. `kennelTags: ["bombay-h3"]` on every event.
5. **Fail-loud guards (both):** `if (runHeadings.length === 0) errors.push(…)` (whole-page drift) **AND**
   per-run — any matched `RUN #NNN` block whose date fails to parse pushes a `ParseError` into `errors[]`
   + `errorDetails.parse` (partial drift must suppress reconcile; see gotchas).

Illustrative shape (control flow only — verify field/import names against the live repo):
```ts
const html = await fetchHTMLPage(url);           // safeFetch under the hood
const $ = cheerio.load(html);
const events: RawEventData[] = [];
const errors: ParseError[] = [];
const blocks = collectRunBlocks($);              // heading /RUN #(\d{3,4})/i → block text, in DOM order
if (blocks.length === 0) errors.push(zeroRowError());
for (const b of blocks) {
  const date = parseBombayDate(b.lines);         // normalize ordinal + chrono, year-bearing, UTC noon
  if (!date) { errors.push(parseError(b.runNumber)); continue; }   // per-run fail-loud
  events.push({
    kennelTags: ["bombay-h3"],
    runNumber: b.runNumber,
    date,
    startTime: parseTime(b.lines) ?? undefined,  // "09:30"
    location: cleanVenue(b.lines) ?? undefined,  // PII-stripped
    // title left undefined → merge synthesizes "Bombay H3 Trail #N"
  });
}
return { events, errors };
```
