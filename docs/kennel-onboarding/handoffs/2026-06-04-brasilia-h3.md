## Implementation gotchas (for Claude Code — repo knowledge, not source knowledge)

  `if (events.length === 0) errors.push("…")` guard so a body-format drift that yields zero parses
  surfaces (a brand-new source's zero-event baseline won't trip the health alert otherwise).

  shortName, safe). Confirm anyway:

- **Dates as UTC noon**, `runNumber` integer, `cuid()` IDs, longer kennel-resolver patterns before
  shorter ones (only `brasilia-h3` here, single-kennel).
- **Sonar S5852 / S5843 / S3776:** keep regexes simple — `RUN_RE`, `DATE_RE` above are single-group,
  no nested quantifiers; resolve the month via a `MONTHS` 3-letter-prefix lookup rather than a
